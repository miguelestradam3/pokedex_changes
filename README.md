# 📖 Pokédex

A modern and responsive **Pokédex** built with **HTML, CSS, Bootstrap 5, JavaScript, and jQuery**. This application allows users to browse Pokémon, search by name or Pokédex number, view detailed information, and explore each Pokémon's stats and types through an intuitive interface.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/miguelestradam3/pokedex_changes.git
```

Open `index.html` in your preferred browser.

> **Note:** If your project loads data from a JSON file using `fetch()`, use a local web server (such as VS Code Live Server). If your data is stored in `pokemons.js`, no server is required.
